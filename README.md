# 📺 Netflix Content Analysis Dashboard | Power BI

## 📌 Project Overview

The **Netflix Content Analysis Dashboard** is an interactive Business Intelligence solution built using **Microsoft Power BI**. It transforms raw Netflix content data into meaningful insights through interactive visualizations, summary cards, and dynamic filters. The dashboard enables users to explore content distribution, ratings, top actors, directors, and yearly content trends in an intuitive and visually engaging interface inspired by Netflix's theme.

---

## 🎯 Objectives

* Analyze Netflix's content library through interactive visualizations.
* Compare the distribution of Movies and TV Shows.
* Identify the most featured actors and directors.
* Evaluate content using IMDb and TMDb ratings.
* Visualize content release trends over the years.
* Present key metrics in a clean and user-friendly dashboard.

---

## 📊 Dashboard Features

* Interactive Movie/Show filter
* Summary cards displaying:

  * Total Titles
  * Total Runtime
  * Average IMDb Rating
  * Average TMDb Rating
* Top Actors Analysis
* Top Directors Analysis
* Movies vs TV Shows Distribution
* Yearly Content Progression
* Netflix-inspired dashboard design
* Interactive and responsive visualizations
